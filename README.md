# CoolDay Environment Setup

Please install the node, python2, jdk and react native first. Also, Please download android studio and configure the ANDROID_HOME environment variable

Version:
react-native-cli: 2.0.1

react-native: 0.62.2

node: v12.12.0


You can follow https://reactnative.dev/docs/environment-setup to setup a React Native CLI environment.



After clone the project to your computer, please follow the steps:
1. open terminal and go to the project folder

2. npm install
```
npm install
```

3. npm audit fix
```
npm audit fix
```

4. Go to android studio, open /CoolDay/android and build the project

5. After building the project, you can start the project
```
npx react-native run-android
```
